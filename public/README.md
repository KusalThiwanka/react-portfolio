## Making Updates

This repository is automatically updated through the deployment process. To make changes:

1. Clone the source repository: `git clone https://github.com/KusalThiwanka/react-portfolio.git`
2. Make your changes in the source code
3. Run `npm run deploy` to build and publish changes
4. Changes will appear here automatically

**Do not edit files in this repository directly** - they will be overwritten on the next deployment.
